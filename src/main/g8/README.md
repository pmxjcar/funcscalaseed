Giter8 seed template for functional libraries
==========================================================================================================================================================================================

Start
--------------------------

1. Install [sbt][sbt]
2. `sbt new pmxjcar/funcscalaseed.g8`
3. `cd $name;format="norm,word"$`
4. `sbt test` (optional)
5. `sbt run`
